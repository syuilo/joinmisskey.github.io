Misskeyをはじめよう

いつまでも進化する、先進的なマイクロブログSNS Misskey（ミスキー）

Misskeyマスコット 藍（Ai）

Misskeyを見つけていただき、ありがとうございます。Misskeyは、地球で生まれた分散マイクロブログSNSです。Fediverse（様々なSNSで構成される宇宙）の中に存在するため、他のSNSと相互に繋がっています。

暫し都会の喧騒から離れて、新しいインターネットにダイブしてみませんか。

はじめる運営する

Misskeyをはじめる理由

カスタマイズ性の高い画面

MisskeyのWebクライアントはあなたの好みに合わせて自在に設定できます。

ホームとデッキ

大きいタイムラインが中央に1つある「ホーム」か、カラムを配置し複数のタイムラインを見られる「デッキ」のどちらかを選んで利用できます。

ウィジェット

小さなウィジェットは様々な情報を表示します。ウィジェットを自由に選んで並べ替えることができます。

自分好みに調整

画面に関する多岐にわたるオプションがあり、操作や表示を細かく変更できます。

多くの機能

Misskeyでできることはたくさんあります。

Wikiで使い方を見る

インスタンスごとに設定された自由な文字数制限

公開範囲を投稿ごとに細かく設定

長い文章や不適切な画像を隠して公開

文字装飾

投稿やプロフィールを装飾しよう。装飾構文(Misskey Flavored Markdown)を利用できます。

MFM構文一覧

リアクション

投稿への共感を簡単に表現しよう。絵文字を選んで、投稿にリアクションを送れます。

アンケート

みんなに意見を聞いてみよう。10個までの選択肢でアンケートを作成できます。複数回答可や有効期限付きにできます。

ドライブ

ファイルを保存して何回も共有しよう。添付したファイルをドライブで管理したり、ドライブにあらかじめファイルをアップロードしたりできます。

ゲーム

みんなで遊ぼう。他のユーザーとゲームを対戦できます。
「リバーシ」はユーザーのアイコンが石として表示されるほか変則ルールでも遊ぶことができます。

独立性と社会性

あなたの意志で運営者を選択する、もしくはあなた自身でサーバーを運営することもできます。

Fediverse

MisskeyはActivityPubというWeb標準規格のサーバー間通信機能を持っているため、 違うインスタンスや他のサービスのユーザーをフォローしたり投稿を見て反応したりすることができます。

簡単なインスタンス構築

Misskeyのインストールは簡単です。自分のSNSを持つことができます。

オープン

Misskeyはオープンソースソフトウェアです。誰でも開発や翻訳に参加することができます。リポジトリをフォークし、独自の拡張を追加することもできます。

GitHubCrowdin(翻訳)

API

アプリやクライアントを作って生活に役立てよう。MisskeyのほぼすべてのAPIエンドポイントを、アプリを作成して利用することができます。

Misskey API
